# **App Name**: VyomGarud Landing

## Core Features:

- Hero Section: Displays the company name, a compelling tagline, a call-to-action, and a strong visual element.
- About Section: Presents a concise overview of VyomGarud's mission and values.
- Capabilities/Products Section: Showcases 3-4 key offerings in an easily digestible card format.
- Highlights Section: Summarizes key features and benefits in three concise bullet points.
- Contact/Footer Section: Provides a simple contact form or relevant links for inquiries.

## Style Guidelines:

- Primary color: Deep charcoal (#333333) for a modern, strong base.
- Background color: Light grey (#E0E0E0), providing contrast to the charcoal elements without being stark white.
- Accent color: Orange (#FF7B00) is used for interactive elements, drawing the eye to calls to action and other areas of focus.
- Font: 'Inter' (sans-serif) for a clean, modern, objective feel. Used for headlines and body text.
- Crisp, minimalist icons that communicate precision and advanced technology.
- Modern, clean layout with a focus on clear hierarchy and readability.
- Subtle CSS or Framer Motion animations for a smooth user experience.
