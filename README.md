# VyomGarud - UAV Systems Landing Page

This is a modern and responsive landing page for VyomGarud Systems, a fictional company specializing in military-grade unmanned aerial vehicles (UAVs). This project was built using Firebase Studio.

## About The Project

This project serves as a comprehensive showcase for VyomGarud Systems, featuring:

*   A dynamic and engaging hero section.
*   Detailed sections for About, Capabilities, and Core Highlights.
*   A functional contact form for user inquiries.
*   A fully responsive design that works across all devices.

## Tech Stack

*   **Framework**: [Next.js](https://nextjs.org/) (with App Router)
*   **Language**: [TypeScript](https://www.typescriptlang.org/)
*   **UI**: [React](https://react.dev/)
*   **Styling**: [Tailwind CSS](https://tailwindcss.com/)
*   **UI Components**: [ShadCN UI](https://ui.shadcn.com/)
*   **Icons**: [Lucide React](https://lucide.dev/)
*   **Generative AI**: [Genkit](https://firebase.google.com/docs/genkit)

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

Make sure you have Node.js (version 18 or higher) and a package manager like npm installed.

```sh
npm install -g npm@latest
```

### Installation & Running Locally

1.  Clone the repository (if you're working outside of Firebase Studio).
2.  Install NPM packages:
    ```sh
    npm install
    ```
3.  Run the development server:
    ```sh
    npm run dev
    ```
4.  Open [http://localhost:9002](http://localhost:9002) with your browser to see the result.

You can start editing the page by modifying `src/app/page.tsx`. The page auto-updates as you edit the file.
